### GET CAT   

#### Это серьезное приложения для получения секретных фотографий котов и кошек с возможностью обнолвения фотографий через каждые 5 секунд.  
  
#### Оглавление
 **- Архитектура проекта:**  
В качестве структуры используется паттерн Atomic Design.  
Страница (Layout) состоит из 1 элемента Компонента Виджета (CheckbBoxWidget) и Компонента Элемента (ImageComponent).   
Реализация функционала обновления фотографий осуществлена с помощью кастомного хука useFetchCat(), который позволяет получить  секретную фотографию рандомной кошки.   

 **- Особенности:**   
 * Для стилизации используются SCSS-модули, которые расположены в папке с компонентами.   
 * Для хранения состояния используется useState.   
    
 **- Установка:**  
 Для запуска приложения необходимо:   
   	Установить зависимости: `npm install`.   
   	Для запуска в режиме разработки: `npm run dev`.
